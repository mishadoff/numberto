# numberto

All you need to play with numbers

### Included

* **numberto.core**
	* `count-digits`
	* `sum-of-digits`
	* `palindrome?` [PRED]
	* `reverse-num`
	* `shift-left`
	* `shift-right`
	* `permutation?`
* **numberto.math**
    * `PI`
    * `E`
    * `power`
    * `power*`
    * `square`
    * `sqroot`
    * `square?` [PRED]
	* `sum`
	* `abs`
	* `avg`
	* `product`
	* `gcd`
	* `lcm`
* **numberto.converters**
    * `char->digit`
    * `digit->char`
    * `num->digits`
    * `digits->num`
* **numberto.predicates**
    * `digit?`
* **numberto.primes**
    * `primes`
    * `prime?` [PRED] 
    * `factorize`
    * `totient`
